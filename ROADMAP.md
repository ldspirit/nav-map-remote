# MVP Roadmap — Africa Digital Map Platform
**Version:** 1.0
**Last Updated:** 2026-02-22

---

## Phase 0 — Foundation (Weeks 1–2)
- Monorepo setup
- Postgres + PostGIS schema
- Auth + OTP
- Address generation core algorithms

## Phase 1 — Vertical Slice MVP (Weeks 3–6)
**Slice 1: Signup → Address → View**
- Address creation (P-system)
- Country format templates
- Basic admin moderation

**Slice 2: Free Navigation**
- Address search
- Route calculation
- Voice guidance

## Phase 2 — Commercial Layer (Weeks 7–10)
- Family plans
- Regional pricing + billing country lock
- Promotions + trials (basic)

## Phase 3 — Trust & Sharing (Weeks 11–14)
- Temporary location sharing
- Basic verification
- Dispute resolution workflow

## Phase 4 — Scale & Optimization (Weeks 15–18)
- Offline cache
- Geo-restricted distribution
- Performance hardening

---

## MVP Exit Criteria
- Users can create a permanent digital address
- P-system works for same-location privacy
- Free navigation available to anyone
- Regional pricing applied at signup
- Basic verification flow operational
